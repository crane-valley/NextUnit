# Speed Comparison Results

The published runtime comparison covers NextUnit, TUnit, MSTest, xUnit, and NUnit, plus Native AOT
variants of NextUnit and TUnit. It uses the same 127-test source suite, native
Microsoft.Testing.Platform executables, and a cyclic round-robin order that balances execution
position across all seven participants.

See:

- [Generated summary](RUNTIME_COMPARISON.md)
- [Raw per-round measurements](runtime-comparison.json)
- [Methodology, controls, and limitations](../../../docs/PERFORMANCE.md)

Reproduce the recorded comparison with:

```bash
cd tools/speed-comparison
dotnet run -c Release --project Tests.Benchmark -- --round-robin 21
```

BenchmarkDotNet reports are complementary diagnostics. The documentation table is generated by the
round-robin mode because running one framework's complete sample set before the next can confound
framework differences with changes in machine load over time.
